+++ date = "2016-02-23T16:06:53+03:00" title = "How to run clean chrome without extensions like the first time" tags = ["tutorials", "chrome"] +++ Hi, guys! If you want to test some page without tons of extension your chrome has, or if you want to check if fresh chrome is 10x faster than chrome with your extensions, bookmarks and settings..
You will need just to create shortcut of chrome and change target, by adding this 
 --user-data-dir=/dev/null
so the target should look like 
"C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --user-data-dir=/dev/null
