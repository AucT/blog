+++ date = "2014-08-03T16:06:53+03:00" title = "How to Remove Directory with Reserved Names Windows" tags = ["articles", "windows"] +++ How to delete folders with reserved names like CON, PRN, AUX, NUL, COM1, COM2, COM3, COM4, COM5, COM6, COM7, COM8, COM9, LPT1, LPT2, LPT3, LPT4, LPT5, LPT6, LPT7, LPT8, and LPT9 in windows??

Here what you need:
1) use cmd (Win+R type cmd and enter)
2) if you have folder: 
RD /s \\.\c:\somedir\com1
if you have file: 
DEL /F \\.\c:\somedir\aux
