  • Name: Invoker Hotkeys
  • Author: AucT
  • Version: 1.07.11
  • Description:

Alt+Ability hotkey will add that ability in pool. Updated 1.07.11 (sendmode input - will work on even if pc has uac)

Note: if it doesn't work - Run it as admin

Guide

  • Alt+Y = Cold Snap
  • Alt+V = Ghost Walk
  • Alt+X = Tornado
  • Alt+C = EMP
  • Alt+Z = Alacrity
  • Alt+D = Chaos Meteor
  • Alt+T = Sun Strike
  • Alt+F = Forge Spirit
  • Alt+G = Ice Wall
  • Alt+B = Deafening Blast

Source Code

Show 
#SingleInstance force
#HotkeyInterval 0
#InstallKeybdHook
#UseHook On
#MaxThreads 20
SetBatchLines, -1
SetKeyDelay , -1, -1
SendMode Input
#IfWinActive Warcraft III
