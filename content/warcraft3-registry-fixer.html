WC3 Registry Fixer - .reg file which will fix registry of your wc3 (replays, worldeditor, wc3 roc, wc3 tft). This time I didn't put it in any program.

Method #1 - Online generate reg file

1. Click generate and customize your path online

Method #2 - Download reg file

1. Click Download Registry Fixer
2. If your WarCraft folder is not C:\Program Files\Warcraft III - edit the file. (use \\ as delimiter)
3. Run Wc3RegistryFixer.reg
